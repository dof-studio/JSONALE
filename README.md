Tired of the human-readable JSON? Here is JSONALE.

JSONALE is `a superset of JSON using binary storage, featureing fast jump-read, indexing, and finer-grained type division. It has a high-performance C implementation and provides C/C++/Python API, supports cross-platform compatibility, conversion to and from JSON, and is most suitable for large non-tabular files.`
